## 背景

熟悉我的人知道，我有过很多个个人网站，其中坚持时间最长的估计就是上一任，用 docusaurus 搭建的网站了（<https://www.cheverjohn.xyz/>）了，整整坚持了六个月有余。我自认为还是很喜欢输出一些文字性的东西的。喜欢将自己所学习的东西分享出去，不断与他人交流，了解一些杂七杂八的东西，不仅限于技术。所以这是我有个人网站的**本心**。
